# Eatko

It's a mobile app helping users to save their time and effort, by recommending recipes for them. Facilitating
the daily question of "What we will eat today?".

## Download App

https://drive.google.com/file/d/1Ha-df3DOnGOde3ycy7SXo9eE2ibGMrHl/view?usp=sharing

## Application Features

 - The application enforces exclusive Left-to-Right (LTR) mode.
 - Efficiently caches images and API calls for improved performance.
 - Manages global data across the application.
 - Implements a specific algorithm to allow users to add favorites to data retrieved from APIs and saves them locally on the device.
 - Incorporates various animations for loading the app, transitioning between screens, and displaying images.

## Packages

 - React-Native
 - Expo
 - TypeScript
 - Redux
 - React-Query
 - Async Storage
 - Redux-Persist
 - React-Navigation
 - React-Animated
